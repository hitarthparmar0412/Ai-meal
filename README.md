# AI Meal - Food Intelligence App

Production-ready Flutter starter for a global AI-powered food intelligence platform.

## Highlights
- Feature-first scalable architecture
- Reusable design system components
- Light + Dark theme tokens
- Bottom navigation with scanner FAB
- Screens for all major product modules and edge cases

## Run
```bash
flutter pub get
flutter run
```

## Structure
`lib/src/features` contains domain features, while `lib/src/core` provides shared widgets/theme/utilities.
